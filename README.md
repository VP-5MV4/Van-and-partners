## О проекте
Одностраничный сайт (лендинг) для юридической компании «Ван и партнеры».

## Технологии
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Supabase (база данных)
- EmailJS (отправка писем)
- AOS (анимации)

## База данных (Supabase)
- Таблица \`categories\` — категории услуг
- Таблица \`services\` — услуги (связь через category_id)

## Форма обратной связи(Emailjs)
- Валидация имени (минимум 2 символа)
- Валидация email (регулярное выражение)
- Toast-уведомления об успехе/ошибке

## Деплой
